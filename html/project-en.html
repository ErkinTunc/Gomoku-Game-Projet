Gomoku Project - Enhanced Organization

Game Description Grid Structure Questions & Extensions

1. Game Description

  • 15x15 Grid, 5 tokens alignment needed to win.
  • 60 tokens each, first move center compulsory.
  • Moves must be adjacent (8-connectivity).

2. Grid Structure

  • Doubly linked matrix with 8 neighbors each.
  • Neighbor updates must maintain reciprocal links.
  • Each cell tracks token presence.
